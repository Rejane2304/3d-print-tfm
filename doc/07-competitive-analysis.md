# Análisis Competitivo - 3D Print TFM

## 🏆 Análisis de Competidores

### Competidor 1: Thingiverse (Marketplace)

**Tipo**: Marketplace de diseños STL (no impresión)

| Aspecto | Evaluación |
|---------|------------|
| **Fortalezas** | Comunidad masiva, diseños gratuitos, tradición |
| **Debilidades** | No venden impresiones, calidad variable, sin soporte |
| **Precio** | Gratis (solo archivos) |
| **Modelo** | Freemium (tips a diseñadores) |
| **Diferenciador** | Nosotros vendemos el producto impreso |

**Amenaza**: Baja - Diferente modelo de negocio
**Oportunidad**: Colaborar con diseñadores de Thingiverse

---

### Competidor 2: Sculpteo (Servicio B2B)

**Tipo**: Impresión 3D industrial profesional

| Aspecto | Evaluación |
|---------|------------|
| **Fortalezas** | Calidad profesional, múltiples materiales, B2B |
| **Debilidades** | Precios elevados, proceso complejo, enfoque industrial |
| **Precio** | Desde 20€ + alta cantidad mínima |
| **Modelo** | B2B fabricación bajo demanda |
| **Diferenciador** | Nosotros: B2C accesible, precios competitivos |

**Amenaza**: Media - Segmento B2B que podría competir
**Estrategia**: Diferenciación por accesibilidad y precio

---

### Competidor 3: Etsy (Marketplace General)

**Tipo**: Marketplace de artículos hechos a mano

| Aspecto | Evaluación |
|---------|------------|
| **Fortalezas** | Tráfico masivo, SEO fuerte, confianza del público |
| **Debilidades** | No especializado 3D, fees altos (6.5%), saturado |
| **Precio** | Variables, competencia ferroz |
| **Modelo** | Marketplace (fees + anuncios) |
| **Diferenciador** | Nosotros especialización 3D + marca propia |

**Amenaza**: Alta - Donde la gente busca productos únicos
**Estrategia**: Diferenciación técnica + calidad superior

---

### Competidor 4: Amazon (Retail)

**Tipo**: Retail generalista

| Aspecto | Evaluación |
|---------|------------|
| **Fortalezas** | Logística, precios, familiaridad, Prime |
| **Debilidades** | Productos genéricos, calidad dudosa, copias |
| **Precio** | Muy competitivo (a veces irracional) |
| **Modelo** | Retail tradicional + marketplace |
| **Diferenciador** | Nosotros: Personalización + exclusividad |

**Amenaza**: Media - No compite en el nicho de personalizado
**Estrategia**: Enfocarnos en lo que Amazon no hace bien

---

### Competidor 5: Makers Locales (Tiendas físicas)

**Tipo**: Makerspaces y tiendas locales

| Aspecto | Evaluación |
|---------|------------|
| **Fortalezas** | Experiencia personal, tocar producto, comunidad |
| **Debilidades** | Alcance geográfico limitado, stock limitado |
| **Precio** | Variable, generalmente más alto |
| **Modelo** | Tienda física + taller |
| **Diferenciador** | Nosotros: E-commerce global + producción propia |

**Amenaza**: Baja - Mercado diferente
**Oportunidad**: Colaboraciones locales

✅ **TDD/MVP**: Desarrollo robusto y probado
✅ **Stack Moderno**: Next.js 14 + Prisma
✅ **Personalización**: Capacidad única

### Debilidades

⚠️ **Sin Historial**: Marca nueva sin reputación
⚠️ **Presupuesto Marketing**: Limitado
⚠️ **Stock Inicial**: Dependencia de producción propia
⚠️ **Tiempo Entrega**: 2-5 días vs Amazon Prime
⚠️ **Conocimiento Brand**: Necesidad de construir awareness

🚀 **Mercado Creciente**: Impresión 3D B2C en expansión
🚀 **Sostenibilidad**: PLA biodegradable, trending topic
🚀 **Personalización Masiva**: Tendencia creciente
🚀 **Nichos Desatendidos**: Productos específicos
🚀 **Makers Community**: Colaboraciones
🚀 **B2B**: Regalos corporativos segmento poco explotado

### Amenazas

⚡ **Competencia Low-Cost**: Productos chinos baratos
⚡ **Recesión Económica**: Bienes no esenciales
⚡ **Rápida Obsolescencia**: Tecnología cambia rápido
⚡ **Dependencia Proveedores**: Filamento, electricidad
⚡ **Regulación**: Normativas productos 3D (futuro)

## 🎯 Estrategia de Diferenciación

### 1. Nichos Específicos

En lugar de competir en todo, enfocarnos en:

| Nicho | Ejemplo Producto | Competencia |
|-------|------------------|-------------|
| Organización escritorio | Soportes cable personalizados | Baja |
| Deco gaming | Soportes mando + auriculares | Media |
| Jardinería urban | Macetas autorriego | Baja |
| Joyería técnica | Gemelos mecanismo reloj | Media |
| Herramientas cocina | Soportes específicos | Baja |

### 2. Ventajas Competitivas Únicas (UVP)

**Propuesta Principal**:
> "Productos impresos en 3D personalizables, de calidad profesional, entregados en 3-5 días, a precios accesibles."

**Pilares**:
1. **Calidad**: Impresiones de calidad profesional (Prusa/Bambu Lab)
2. **Personalización**: Modificamos diseños según necesidad
3. **Velocidad**: Producción local, envío rápido España
4. **Precio**: Sin intermediarios, producción propia
5. **Soporte**: Chat directo, atención personalizada

### 3. Posicionamiento de Precio

```
Precio
   │
   │   💰 Premium
   │      (Sculpteo)
   │
   │   🎯 NOSOTROS
   │      (Calidad/Precio)
   │
   │   🏪 Mass Market
   │      (Amazon/Etsy)
   │
   │   💸 Low Cost
   │      (AliExpress)
   └─────────────────── Calidad
```
